# Implementation Plan: 豆寄席プレゼンテーション資料作成

**Branch**: `001-mameyose-presentation` | **Date**: 2026-03-05 | **Spec**: [001-mameyose-presentation/spec.md](spec.md)

## Summary

本計画書は、「豆寄席」で発表予定の90分枠（発表80分＋Q&A10分）のプレゼンテーションスライド作成のための構成案を示すものです。
スクラムマスターを主なターゲットとし、AIを活用した経験的プロセス制御（透明性・検査・適応）の強化手段を具体的な事例（Jira×GAS×AIでの可視化、AIによるレトロスペクティブ評価）を通して解説します。また、最後に当スライドの作成自体をAIエージェント（Antigravity）やGitHub Spec Kit、Marpを用いて行った裏側（メイキング）を紹介します。

## Technical Context

**形式**: `Marp for VS Code` (MarkdownからスライドPDF/HTMLを出力)
**デザインシステム**: GitHub Primer CSSのカラースキームと構成に準拠、Octicons使用
**ベースカラー**: 薄い紫色 (`#F6F3FB` など)  
**スライドテンプレ機能**: 右上に会社ロゴ（`mamezou.png`）固定配置、フッターにページ番号のみ  
**フォント**: システムゴシック（Apple System等）ベース、日本語は可読性の高い `Noto Sans JP` を想定。最低 24pt を確保。
**使用生成AIツール**: Antigravity, GitHub Spec Kit
**CI/CD**: GitHub Actionsを用いたMarpスライドのPDF/HTML自動ビルドとGitHub Pagesへの自動デプロイ

### 3. 第2部：透明性の強化 (約25分 / スライド約15〜20枚)
